# H 奇美鲁斯，未梦之神
> 副本：梦境裂隙
> 难度：英雄
> 维护说明：后续请优先直接更新这份 Markdown，我会按这份结构同步回 JSON。

## 战斗摘要

### 一句话

一场在内外场之间循环切换、处理裂隙具象与飞天阶段深呼吸的双阶段战斗。

### 战斗类型

双场地轮换 + 内外场小怪处理 + 双阶段长轴战斗

### 击杀条件

P1 稳定分组进入裂隙处理具象并防止外场小怪被 Boss 吞噬，P2 规避深呼吸并清掉新增具象，最终在循环中压低 Boss 血量。

## 开荒速览

### Boss 站位

P1 外场 Boss 需正面朝向远离大团，避免猛撕开裂劈到无关目标；P2 通过集合站位引导深呼吸路径。

### 优先目标

- 裂隙内需要尽快破盾送出的具象
- 外场正在靠近 Boss、即将被吞噬的小怪
- P2 深呼吸后刷新出的高危具象
- 需要驱散吞噬瘴气并顺手消水的目标

### 核心循环

1. P1 起手与 50 能量触发裂隙涌现，内场处理具象并送回外场
2. 15 能量和 65 能量的艾林之尘剧变分两组把玩家送入内场轮换处理
3. 外场优先击杀已送出的具象，避免被 Boss 吞噬叠增伤并回血
4. 满能量后进入 P2，按深呼吸路径移动并清理新刷具象

### 治疗压力点

- 裂隙涌现与内场具象刷新后的群体压力
- 腐蚀黏痰的固定环境伤害
- 吞噬瘴气驱散前后的蓝圈爆发
- P2 深呼吸、贪食俯冲和新增小怪并行时的大团掉血

### 常见灭团点

- 内场具象处理过慢，外场小怪被 Boss 吞噬
- 吞噬瘴气没有靠橙水驱散，导致场地被持续压缩
- 深呼吸引导失误，P2 场地被多条路径切碎
- P2 结束前场上仍有大量具象存活，被 Boss 落地时直接吞掉

## 职责提示

### Tank

职责定位：负责吃分摊进内场、拉稳外场具象，并在 P2 维持 Boss 朝向与聚怪秩序。

- 艾林之尘剧变的分摊圈要按预定分组轮流进入内场。
- 外场大怪会有不谐咆哮与巨像打击，注意减伤与换坦节奏。
- P2 深呼吸后新刷小怪要尽快聚拢，避免外场失控。

### Healer

职责定位：覆盖内外场并行伤害，处理吸收盾、驱散与高压深呼吸阶段。

- 裂隙涌现和内场小怪出现时是首波治疗高峰。
- 吞噬瘴气驱散前后要重点照顾被点目标及其周围分摊人群。
- P2 深呼吸与贪食俯冲后全团位置容易分散，要预留机动抬血。

### DPS

职责定位：优先处理具象与卵，控制外场存活目标数量，并在 P2 快速清掉新增小怪。

- 内场先破盾把具象送出，再打 Boss。
- 外场小怪永远高于 Boss 优先级，不能让其被吞噬。
- P2 深呼吸后优先处理高威胁具象并为大团让路。

## 技能详解

### 艾林之尘剧变

- 分类：分组换场
- 严重度：核心机制

奇美鲁斯对当前坦克施放分摊圈，被命中的玩家会被击飞进入内场并获得一段时间的艾林洞察。由于裂隙易伤持续较长，内外场必须提前分组轮换进入。

Tank：
- 15 能量和 65 能量两次分摊都要按既定分组执行。

Healer：
- 进内场的那一组短时间会脱离外场治疗覆盖，需要预铺。

DPS：
- 明确自己属于哪一组，别在错误轮次进内场。

![艾林之尘剧变分摊进内场示意](../../assets/3_d17d1e6f.gif)

### 裂隙涌现 / 具象处理

- 分类：内场核心
- 严重度：核心机制

P1 起手和 50 能量会触发裂隙涌现，在内场刷出多种具象。内场具象有护盾，打破后会被送回外场并留下艾林之尘精华。

Tank：
- 进内场后优先稳定大怪位置，避免连击打散队伍。

Healer：
- 关注融合具象带来的吸收治疗效果。

DPS：
- 先破盾送怪，再打 Boss；别在内场逗留过久。

![裂隙涌现与内场刷新示意](../../assets/3_ce2b8b72.gif)
![内场组合整理图](../../assets/3_5d821ag.webp)

### 巨身憎恶

- 分类：高危具象
- 严重度：坦克压力

大怪会周期性 AOE 并对当前坦克发动高额三连击，是内外场都需要优先控制的大型目标。

Tank：
- 提前减伤应对巨像打击。

Healer：
- 不谐咆哮叠层后团队压力会快速上升。

DPS：
- 需要尽快转火，避免大怪在场上存活过久。

![巨身憎恶示意](../../assets/3_3f6c00da.gif)

### 萦绕的精华

- 分类：可打断具象
- 严重度：打断关键

中怪会读可怖战吼和精华箭矢，若不打断会造成恐惧和连续伤害。

Tank：
- 方便近战贴身打断即可，不必拉太远。

Healer：
- 恐惧一旦放出会造成全团连锁减员风险。

DPS：
- 提前分配打断链，避免同一波多只同时读条。

![萦绕的精华示意](../../assets/3_5de06abu.gif)

### 吞噬 / 被吞噬的精华

- 分类：灭团校验
- 严重度：灭团技

奇美鲁斯会在满能量前后吞噬场上的具象。当前版本下，任何具象被吃都可能导致回血、全团伤害和大幅增伤，因此外场具象优先级极高。

Tank：
- 保证所有具象都被拉离 Boss 吞噬路径。

Healer：
- 吞噬前后要准备全团抬血。

DPS：
- 外场小怪 > Boss，本场最重要的输出原则就是别让怪被吃。

![吞噬与吃怪示意](../../assets/3_52317a5b.gif)
![三类具象控制参考](../../assets/3_7374fdr.webp)

### 吞噬瘴气

- 分类：驱散与消水
- 严重度：场地处理

P1 外场以及 P2 部分阶段会点名玩家套吞噬瘴气。驱散时会在玩家周围爆发，并可顺带消除蓝圈覆盖到的艾林之尘精华。

Tank：
- 被点名时提前远离 Boss 正面和近战位。

Healer：
- 驱散前确认目标已经到位，避免炸团。

DPS：
- 被点后主动靠橙水等待驱散，别把爆炸带进人群。

![吞噬瘴气消水示意](../../assets/3_3ec3e5ch.gif)

### 猛撕开裂

- 分类：正面规避
- 严重度：站位约束

外场 Boss 会朝前方锥形区域顺劈并附带流血，所有非当前目标都要离开正面。

Tank：
- 保持 Boss 朝向稳定。

Healer：
- 正面误吃的人会有持续流血，需要快速补救。

DPS：
- 永远别站 Boss 头前贪输出。

![猛撕开裂示意](../../assets/3_68f46f84.gif)

### 腐化毁灭 / 贪食俯冲

- 分类：P2 核心
- 严重度：阶段机制

飞天阶段 Boss 会按玩家位置喷深呼吸，并在阶段结束前俯冲落地。每次深呼吸都会带来新的具象与场地橙水，阶段结束时若仍有怪存活会被直接吞噬。

Tank：
- 用集合站位帮助团队统一引导深呼吸。

Healer：
- 深呼吸与贪食俯冲都是团队大伤时点。

DPS：
- P2 以清怪与给大团让路优先，不要只打 Boss。
