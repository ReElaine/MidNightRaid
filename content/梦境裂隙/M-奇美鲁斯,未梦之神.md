# M 奇美鲁斯，未梦之神

> 副本：梦境裂隙
> 难度：史诗
> 维护说明：本篇以史诗相对英雄的差异点为主，完整基础机制可直接参照同目录下的英雄攻略。

## 战斗摘要

### 一句话

史诗难度在英雄框架上加入内外场互炸、裂隙疯狂救人换位，以及 P2 内外场同步刷怪，整体更强调站位纪律和换位执行。

### 战斗类型

英雄本体机制强化 + 内外场互炸 + 救人换位 + 双场同步清怪

### 击杀条件

维持内外场固定站位，稳定处理裂隙疯狂和救人换位；P2 同步清掉内外场新增具象，并避免任何目标被 Boss 吞噬。

## 开荒速览

### Boss 站位

延续英雄难度本体站位，但必须额外区分内外场固定点位，避免不谐导致跨位面互炸。

### 优先目标

- 被点裂隙疯狂的内场玩家
- 负责救人的外场对应角色
- P2 内外场同时刷新的新增具象

### 核心循环

1. 战前分好内外场固定站位，严格规避不谐互炸
2. 每轮分摊后处理裂隙疯狂，并由外场预设救人组第一时间换位
3. 满能量前额外吃一次艾林之尘剧变，再转入 P2
4. P2 内外场同步刷怪，双线同时清理

### 治疗压力点

- 不谐带来的跨位面周期互炸
- 裂隙疯狂点名后的持续高压掉血
- P2 双场同时刷怪时的并行压力

### 常见灭团点

- 内外场站位混乱，触发连锁不谐爆炸
- 救人组进圈不及时，裂隙疯狂目标被拖死
- P2 双场新增具象处理脱节，一侧先崩盘

## 职责提示

### Tank

职责定位：在英雄换坦和吃圈基础上，维持内外场站位稳定，并保证 P2 转阶段安全。

- 英雄本体的分摊和换坦节奏仍然成立。
- P2 飞天前新增的艾林之尘剧变需要提前安排分摊对象和减伤。
- 内外场切换时不要把仇恨和聚怪路径拉乱。

### Healer

职责定位：覆盖不谐互炸、裂隙疯狂与 P2 双场刷怪的复合压力。

- 裂隙疯狂点名阶段要提前预留大治疗与减伤。
- 救人换位后，两边治疗覆盖对象会立刻变化，需要快速调整。
- P2 双线处理时要明确每名治疗主要跟哪一边。

### DPS

职责定位：严格执行内外场站位纪律，并在救人与清怪阶段保持高效处理。

- 不谐要求你明确自己属于内场还是外场。
- 被点裂隙疯狂时按预定光柱站位，不要把惊骇扩散给周围队友。
- P2 双场刷怪时仍然以清怪为最高优先级。

## 技能详解

### 不谐

- 分类：位面约束
- 严重度：史诗核心变化

内外场玩家在 8 码内会周期性互相造成自然伤害，因此战斗开始前就必须把两组人的固定站位完全拆开。任何跨位面贴近都会迅速积累治疗压力。

Tank：交接和换位阶段不要把两边玩家拉到同一区域。  
Healer：站位一旦混乱，团队血线会被很快压穿。  
DPS：把固定站位当成刚性规则执行，不要临场乱跑。

### 裂隙疯狂

- 分类：救人换位
- 严重度：史诗核心变化

P1 每轮分摊后，Boss 会点名内场 1 奶 + 1 DPS 进入裂隙疯狂。被点目标会在短时间后惊骇并持续猛掉血，需要由外场预设的 1 奶 + 1 DPS 第一时间进圈换位把人救出来。

- 团测里通常会在场地中间放两个光柱给被点目标固定站位。
- 救人后，救人的玩家会进入内场，被救的玩家回到外场。
- 换位时绝对不能跑错半场，否则会立刻引发不谐问题。

Tank：保证救人圈周围路径畅通，不要被 Boss 或大怪卡位。  
Healer：被点目标要吃自保和点刷，外场治疗要准备接应救人组。  
DPS：救人组必须第一时间进圈，不能拖到惊骇完全炸开。

### P2 前额外艾林之尘剧变

- 分类：转阶段强化
- 严重度：史诗新增

史诗难度下，Boss 满能量后不会直接飞天，而是先完成吞噬引导，再额外施放一次艾林之尘剧变，最后进入 P2。这个变化会把转阶段节奏拉长，也要求团队额外安排一次分摊进场。

- 这次剧变之后，P2 内场也会对应多刷一组具象。
- 转阶段前要先把分摊站位和 P2 清怪责任讲清楚。

Tank：提前安排吃圈对象与减伤。  
Healer：吞噬结束后紧接分摊，治疗节奏很紧。  
DPS：先把转阶段分摊与站位处理完，再准备 P2。

![P2 前额外分摊示意](../../assets/4_bbdcf27h.gif)

### P2 双场刷怪

- 分类：阶段强化
- 严重度：史诗新增

史诗难度的 P2 不再只是外场处理新增具象，而是内外场各自都会刷新一组怪。团队需要同步处理两边压力，任何一边崩盘都会让 Boss 迅速吞噬剩余目标。

- 根据已有测试，第一次深呼吸对着外场组，第二次对着内场组，第三次再回到外场组。
- 刷新的怪物种类与英雄相同，但同时存在于双场，容错显著下降。

Tank：两边都要有明确的接怪和减伤安排。  
Healer：不要把所有资源压在单边，P2 是双线掉血。  
DPS：优先稳住两边清怪，再考虑 Boss 伤害。
