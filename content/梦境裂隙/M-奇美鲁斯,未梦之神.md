# M 奇美鲁斯，未梦之神
> 副本：梦境裂隙
> 难度：史诗
> 维护说明：本篇目前整理史诗差异点为主，后续可继续按完整模板补齐。

## 战斗摘要

### 一句话

史诗难度在英雄框架上增加了内外场互炸、救人换位和 P2 内外双场刷怪，整体更强调站位纪律与换位执行。

### 战斗类型

英雄本体机制强化 + 内外场互炸 + 救人换位 + 双场刷怪

### 击杀条件

维持内外场固定站位、稳定处理裂隙疯狂、在 P2 同步清掉内外场新增具象，并避免任何具象被 Boss 吞噬。

## 开荒速览

### Boss 站位

延续英雄本体站位，但必须额外区分内外场固定站点，避免不谐互炸。

### 优先目标

- 被点裂隙疯狂的内场玩家
- 负责救人的外场对应角色
- P2 内外场新增具象

### 核心循环

1. 维持内外场固定站位，规避不谐互炸
2. P1 每轮分摊圈后处理裂隙疯狂并完成救人换位
3. P2 飞天前额外吃一次艾林之尘剧变
4. P2 期间内外场各自处理新增具象

### 治疗压力点

- 不谐造成的跨位面周期互炸
- 裂隙疯狂点名后的高压掉血
- P2 同时两边刷怪时的持续团队伤害

### 常见灭团点

- 内外场站位混乱导致不谐连锁爆炸
- 救人组进圈不及时，裂隙疯狂直接拖死人
- P2 内外场新增具象处理断档

## 职责提示

### Tank

职责定位：在英雄本体换坦基础上，维持两边站位稳定并保障 P2 转阶段安全。

- 英雄本体的分摊与换坦仍是基础。
- P2 飞天前新增的艾林之尘剧变需要提前规划站位与减伤。
- 内外场切换时尽量不要让仇恨和聚怪路线混乱。

### Healer

职责定位：覆盖不谐互炸、裂隙疯狂与 P2 双场刷怪的复合压力。

- 裂隙疯狂点名阶段要提前预留大治疗和减伤。
- 救人换位后，两边治疗覆盖对象会瞬间变化，需及时调整。
- P2 内外场双线处理时要明确哪个治疗跟哪边。

### DPS

职责定位：严格执行内外场站位纪律，并在救人与清怪阶段维持高效率。

- 不谐要求你明确自己属于内场还是外场组。
- 被点裂隙疯狂时按预定光柱站位，别把惊骇传给周围人。
- P2 双场刷怪时先处理怪，再考虑 Boss 伤害。

## 技能详解

### 不谐

- 分类：位面约束
- 严重度：史诗核心变化

内外场玩家在 8 码内会周期互炸，因此战斗前必须把两组固定站位完全分清。

Tank：
- 交接阶段不要把两边玩家拉到同一区域。

Healer：
- 任何跨位面站位失误都会快速积累治疗压力。

DPS：
- 把自己的站位当作刚性规则执行，别临场乱跑。

### 裂隙疯狂

- 分类：救人换位
- 严重度：史诗核心变化

P1 每轮分摊后会点内场 1 奶 + 1 DPS 裂隙疯狂。被点者会惊骇并持续猛掉血，需要外场预定救人组第一时间进圈换位救人。

Tank：
- 让救人圈位置保持可达，不要被大怪或 Boss 站位卡死。

Healer：
- 被点目标要吃自身减伤，内场治疗要点刷，外场治疗负责接应救人组。

DPS：
- 救人组必须第一时间进圈，不要拖到惊骇完全炸开。

### P2 艾林之尘剧变

- 分类：转阶段强化
- 严重度：史诗新增

史诗难度中，Boss 在飞天前会额外施放一次艾林之尘剧变，进一步打乱转阶段节奏。

Tank：
- 这次分摊要提前安排好吃圈对象。

Healer：
- 吸吸结束后立刻接分摊，治疗节奏会非常紧。

DPS：
- 转阶段前先把分摊和站位处理完，再准备 P2 清怪。

![P2 前额外分摊示意](../../assets/4_bbdcf27h.gif)

### P2 双场刷怪

- 分类：阶段强化
- 严重度：史诗新增

史诗难度的 P2 会在内外场各自新增一组具象，团队需要同时处理两边压力。

Tank：
- 两边都要有明确接怪与减伤安排。

Healer：
- 别把所有资源压在单边，P2 是双线掉血。

DPS：
- 优先稳定两边清怪，不要让任何一边崩盘。

![史诗 P2 刷怪整理图](../../assets/4_6da4d36.webp)
